# Manual de Etiqueta e Segurança do Estudante

Este repositório apresenta o **Manual de Etiqueta e Segurança do Estudante**, desenvolvido como meu **Projeto Integrador** para a conclusão do curso:

**Assistente de Tecnologias da Informação**  
SENAC – Barretos (Brasil)
Concluído com êxito no ano de 2025.

## 📘 Sobre o Projeto

O manual foi criado com o objetivo de orientar estudantes quanto à convivência respeitosa, uso adequado dos espaços e práticas de segurança institucional.

Baseia-se em diretrizes educacionais e promove:
- Convivência harmoniosa
- Postura profissional
- Segurança individual e coletiva

## 🎯 Objetivo do Repositório

O principal objetivo deste repositório é demonstrar as habilidades utilizadas no desenvolvimento do projeto, tais como:

- 🎨 Design gráfico  
- 🔎 Pesquisa e organização de conteúdo  
- 💻 Habilidades técnicas (criação de documento digital e publicação online)
